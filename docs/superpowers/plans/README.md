# Implementation Plans

Implementation plans live in this directory.

Before building a feature:

1. Read the relevant spec in `docs/superpowers/specs/`.
2. Create a plan named `YYYY-MM-DD-<feature-name>.md`.
3. Include explicit file paths, test strategy, verification commands, and commit checkpoints.
4. Keep tasks small enough for agentic execution.
